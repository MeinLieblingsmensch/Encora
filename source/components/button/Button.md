
Default button:

```js
<Button>
    Push Me
</Button>
```

Button with dynamic background and font color:

```js
<Button
    background='green'
    color='white'
>
    Push Me
</Button>
```


A rounded button

```js
<Button
    rounded
>
    Push Me
</Button>
```

```js
<Button
    rounded='50'
>
    Push Me
</Button>
```

A fluid button

```js
<Button
    width='fluid'
>
    Push Me
</Button>
```

## Font

Upper case font styling

```js
<Button
    uppercase
>
    Push Me
</Button>
```

Bold font

```js
<Button
    bold
>
    Push Me
</Button>
```
